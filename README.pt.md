## Por que existe

TacticalPad e similares sao otimos mas sao pagos, antigos ou complicados demais. **TacticBasicsFootball** e a resposta: gratis, open source, moderno e pensado para um treinador pegar um tablet no campo e usar sem tutorial.

## Quick start

```bash
git clone https://github.com/meser1905/tactic-basics-football.git
cd tactic-basics-football
pnpm install
pnpm dev
```

Abra [http://localhost:3000](http://localhost:3000) e pronto.

## Stack

Next.js 15, React 19, TypeScript, Tailwind CSS v4, shadcn/ui, Three.js, React Three Fiber, Zustand, Vitest, Playwright.
